[Tor] MacOSでTorの簡単なインストール方法

2017年1月10日 開発, Tor

MacOSで簡単にTorをインストール方法についてノートする。 Mac Sierraでテストしましたが、他のMacでもできると思います。

1. Torインストール

# Brewの場合
$ brew install tor

# Macportsの場合
$ sudo port install tor

2. Tor起動スクリプト準備

tor.shなど好きな名前でスクリプトを以下の内容で作成して実行権を与えます。

3. 起動スクリプト実行

$ ./tor.sh

詳しくは Running Tor on Mac OS Xをご参考ください。.
