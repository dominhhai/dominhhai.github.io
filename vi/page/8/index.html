[Git] submodule - cùng lúc sài nhiều repo

3 tháng 9, 2017 mục Lập Trình, Git

Ở đây, mình sử dụng máy chủ CentOS 7 và Nginx để thực hiện, nhưng các bạn hoàn toàn có thể làm tương tự với các môi trường khác.
